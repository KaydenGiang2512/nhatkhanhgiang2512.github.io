WHAT IS GITHUB?

  Github Inc. was founded on February 8th 2008 by the company Logical Awesome LLC, but has recently become a subsidiary of Microsoft since 2018. 
  The best thing about using Github is that it offers it basic services completely free-of-charge, though if you need to gain access to more advanced and professional enterprise services, you might need to subscribe to a monthly plan.

see plans here

  Basically, Github is an online repository hosting service for software development and version control using the Git VCS.
  Apart from the fuctionalities it already inherits from Git, Github is also equipped with some additional features which helps a great deal with managing, collaborating and sharing codes for group projects.

Notable features

Personally, I found Github to be an extremely helpful tool that outclasses other similar platforms of its kind, and below are just 6 reasons why Github is the best: 

"Forking"

This is the flagship functionality of Github - being able to copy a repo between different accounts, then modify the files to your own preference and finally sharing it back via the "pull request" notification. This feature significantly increases individual productivity as well as teamwork. 

Open source-friendly

Being a free online platform specifically for managing open source projects and hosting websites, Github makes contributing to your works much easier and including documents or getting feedback also becomes more convenient.

Well-padded documentation

Github proudly owns one of the most refined and in-depth documentation out there. The content you will receive by using Github is excellent, and its coverage is virtually anything you could possibly think of regarding Git.  

Showcase your work

This platform allows you to have your personal online profile, which would greatly improve your chances of getting in touch with a recruiter/scout, and potentially landing your ideal job in the foreseeable future.

Simplified data backup

Another feature that Github provides for its users is the ability to save their progress and version history online, regardless of what they do on their local machines.

Integration options

Not only can Github function independently, it can integrate with other common platforms such as Amazon/Google Cloud, and services like Code Climate for syntax checking and tracking feedbacks.
